# dues
The cloudtag's due system.

## Instalation:

Clone this repo, open a terminal, go to the repo's folder and run the following commands:

1: Install all the gems we need.
 `bundle install`

2: Create the db.
`rake db:migrate`

3: (optional) Insert some seeds.
`rake db:seed`

And that's it!

Just run rails using  `rails s`

Enjoy!
